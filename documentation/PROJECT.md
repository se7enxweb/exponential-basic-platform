Exponential Basic 2.4.0.1
----------------

Exponential Basic is a content management system for e-commerce, e-publishing and intranets.

The software is licensed under the GPL as described in the file [../LICENSE.md](LICENSE.md).

For the changelog consult the file [CHANGELOG](CHANGELOG).

For the credits (who made what) consult the [CREDITS](CREDITS) file.

For information about how to translate Exponential Basic consult the file [TRANSLATION](TRANSLATION)

For installation information consult the file [INSTALL.txt](INSTALL.txt)

NOTE: There are structural changes in the database that might make your old articles unavailable when you upgrade. 
There are some SQL sentences in the [SQL_UPDATES](SQL_UPDATES) file to fix this. To fix please read the comments.
